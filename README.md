# 🌸 Miral Alsharief — Personal Portfolio

> A personal portfolio website showcasing my projects, skills, and journey as a Computer Science student.

---

## ✨ Features

- **Dark Neon Design** — Pink & green neon aesthetic with scanline effects
- **Custom Cursor** — Glowing cursor effect
- **Smooth Animations** — Fade-up transitions on scroll
- **Fully Responsive** — Works on desktop and mobile
- **Sections** — About, Skills, Projects, Contact

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone <https://github.com/miral515/Miral-Portfolio.git>
   ```

2. Open the project folder.

3. Create an `img` folder and add the required assets:
   - `glow-cursor.cur`
   - `flower-store-preview.png`
   - `MiralAirCanvas.png`

4. Open `index.html` in your browser — no setup needed!
